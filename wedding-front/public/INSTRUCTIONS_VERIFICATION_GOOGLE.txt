========================================
COMMENT VÉRIFIER VOTRE SITE AVEC GOOGLE
========================================

PROBLÈME : Vous ne trouvez pas le fichier de vérification Google ?

SOLUTION : Vous devez le télécharger depuis Google Search Console !

ÉTAPES À SUIVRE :
================

1. Dans Google Search Console, cliquez sur "Paramètres" (roue crantée en bas à gauche)

2. Cherchez la section "Vérification de la propriété" ou "Ownership verification"

3. Cliquez sur "Fichier HTML" dans les méthodes de vérification

4. Google va vous proposer un fichier à télécharger avec un nom comme :
   googleXXXXXXXXXXXX.html
   
5. TÉLÉCHARGEZ ce fichier sur votre ordinateur

6. PLACEZ ce fichier dans ce dossier : 
   wedding-front/public/
   
7. Le fichier doit être accessible à :
   https://kawepla.kaporelo.com/googleXXXXXXXXXXXX.html

8. Retournez sur Google Search Console et cliquez sur "Vérifier"

MÉTHODE ALTERNATIVE (PLUS SIMPLE) :
===================================

Utilisez plutôt la méthode "Balise HTML" :

1. Dans Google Search Console, choisissez "Balise HTML"

2. Copiez le code qui ressemble à :
   <meta name="google-site-verification" content="abc123xyz456" />

3. Ouvrez le fichier : wedding-front/src/app/layout.tsx

4. Cherchez "export const metadata"

5. Ajoutez cette ligne :
   
   verification: {
     google: 'abc123xyz456',  // Votre code ici
   },

6. Déployez votre site

7. Vérifiez sur Google Search Console
